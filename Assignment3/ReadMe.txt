README file for sortComp/sortPrint/heapDriver ADT
NOTE: use “[program to run] [test file]” to run the program
The sortComp ADT contains the following files:
ReadMe.txt 	This documentation file
heap.h		The header file describing the adt operations for heap.c
heap.c		The implementations of the heap ADT
insertionSort.h The header file describing the adt operations for insertionSort.c
insertionSort.c The implementations of the insertion ADT
heapDriver.c    A simple program that tests out the ADT heap operation.
heapSort.h        The header file describing the adt operations for heapSort.c
heapSort.c         The implementation of the heapSort ADT
sortPrint.c          The program that prints out the sorted values of heapSort and insertionSort
sortComp.c         The program that prints out the time run for heapSort and insertionSort
Makefile             A makefile that complies the module and driver for sortPrint, sortComp, 
		 heapDriver
